# Identificação de Sistemas

Notas de ensino e pesquisa em Identificação de Sistemas: ajuste de reta por
mínimos quadrados, simulação de sistemas e sinais PRBS, identificação de
modelos ARX e identificação de um sistema Linear Invariante no Tempo (LIT)
a partir de dados de entrada/saída.

🔗 Site: https://raphateixeira.github.io/Identificacao/

## Conteúdo

- `Identificacao/Ident01AjusteReta.qmd` — ajuste de reta por mínimos quadrados.
- `Identificacao/Ident02SimulacaoPRBS.qmd` — simulação de sistemas e sinais PRBS.
- `Identificacao/Ident03ARXMinimosQuadrados.qmd` — identificação de modelos ARX por mínimos quadrados.
- `Identificacao/Ident04IdentificacaoLIT.ipynb` — identificação de um modelo LIT via PRBS e mínimos quadrados, com avaliação do ajuste (R²). Migrado do antigo repositório `IdentificaLSTM`.

Uma segunda etapa de identificação via LSTM foi cogitada no antigo
`IdentificaLSTM`, mas nunca chegou a ser desenvolvida — fica como trabalho
futuro possível aqui.
